# Java files for the app

This BMI calculator contains two activities/pages.
1.content_main.xml
2.content_main2.xml

#MainActivity.java
It is a java file for the content_main.xml which include backend java code which defines the functionality of the
different views and button used in this content_main activity.

#Main2Activity.Java
It is a java file  which include java code for the content_main2.xml which fetch the response from the content_main.xml and show the proper structured result on content_main2.xml.

#MyDbHandler.java and Product.java
These files includes java code which implement squlite database and connect the activity to the data base.

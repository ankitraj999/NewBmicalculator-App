# Java files for the app

This BMI calculator contains two activities/pages.
1.MainActivity.main
2.Main2Activity.main

#MainActivity.java
It is a java file for the MainAcitivity.main which include backend java code which defines the functionality of the
different views and button used in this MainActivity.

#Main2Activity.Java
It is a java file for the Main2Activity which include java code for the the Main2Activity.main which fetch the response from 
the MainActivity and show the proper structured result.
